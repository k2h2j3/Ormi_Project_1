# Movie Sommelier(무비 소믈리에)

## 기능
### 1. 사용자의 취향을 입력받아 챗봇이 영화를 추천해준다.
### 2. 최신 영화와 베스트 영화의 줄거리를 간략하게 알려주고(modal 사용) 예고편링크로 연결해준다.
